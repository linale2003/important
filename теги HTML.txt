1)<p>-парный тег для параграфа
2)<strong>-парный тег для выделения жирным шрифтом
3)<article>-парный тег(независия отдельная часть контента,которая имеет свой заголовок и содержание,которое можно без проблем взять и использовать на другом сайте)
4)<section>-парный тег,Часть документа,которая имеет свой смысл и значение.От article отличается невозможностью вырывать со страницы или контекста страницы(заголовки <h2><h3>)
5)<nav>-навигационные разделы сайта,то есть разные меню
6)<header>-вводная часть отдельного раздела(article или section),обычно используется для подсказок,анонсов,навигации
7)<main>-Основное,не повторяющееся на других страницах содержимое.Он один на страницу,а внутри идут другие блоки.Такое себе оберточнцый контейнер
8)<footer>-нижняя(подвальная или футерная)часть сайта,которая в большинстве случаем не изменена на всех страницах
9)<table>-одиночный тег для таблицы
10)<div>-парный тег,для универсального контейнера без какой-либо логической составляющей .Применяется для самых разных целей
11)<img>-контентное изображение
12)<a>-ссылка
13)<button>-кнопка
14)<br>-перенос строки
15)<span>-универсальный фазовый элемент без собсвтенного значения
Атрибуты:
16)<accesskey>-Атрибут для горячих клаввиш
синтаксис:<accesskey="s"> для разных браузеров своя доп. клавиша
17)<contenteditable="true">-атрибут,который включает режим редактирования
18)<contextmenu>-атрибут,В качестве значения указывается идентификатор меню, созданного с помощью тега <menu>.
19)<dir>-атрибут.Задает направление и отображение текста — слева направо или справа налево. Браузеры обычно самостоятельно различают направление текста, если он задан в кодировке Юникод, но с помощью атрибута dir можно указать, в каком направлении отображать текст. Для арабских и еврейских символов приоритетным является направление, заложенное в Юникод, поэтому на них атрибут dir действовать не будет.
dir={ltr | rtl}
Значения
ltr
Текст отображается как обычно — слева направо.
rtl
Текст инвертируется и отображается справа налево.

20)<hidden>-атрибут,который скрывает содержимое элемента от просмотра.Такой элемент не отображается на странице,но доступен через скрипты
21)<id>-атрибут,который Задает стилевой идентификатор — уникальное имя элемента, которое используется для изменения его стиля и обращения к нему через скрипты. Идентификатор в коде документа должен быть в единственном экземпляре, иными словами, встречаться только один раз.

Синтаксис
id="имя"

22)<lang>-атрибут,Текст документа может быть набран как на одном языке, так и содержать вставки на других языках, которые могут различаться по своим правилам оформления текста. Например, для русского, немецкого и английского языка характерны разные кавычки, в которые берется цитата. Чтобы указать язык, на котором написан текст внутри текущего элемента и применяется атрибут lang. Браузер использует его значение для правильного отображения некоторых символов.

Синтаксис
lang="код языка"
23)<spellcheck>-атрибут,который указывает проверять или нет правопписание и граматику в тексте.
Синтаксис
spellcheck="true | false"
Значения
true
Включает режим проверки правописания.
false
Отключает проверку.
24)<style>-парный тег,который применятеся для определения стилей элементов вею-страницы.Тег<style> необходимо исползовать внутри контейнера<head>.можно задавать более чем один тег <style >
Синтаксис
<head>
  <style type="text/css">
   ...
  </style>
</head>
Атрибуты
media
Определяет устройство вывода, для работы с которым предназначена таблица стилей.
type
Сообщает браузеру, какой синтаксис использовать, чтобы правильно интерпретировать сти
25)<tabindex>-атрибут который опрдееляет последовательность перехода между ссылками при нажатии на кнопку Tab.
Синтаксис
<a tabindex="число">...</a>

26)<title>-атрибут ,который Определяет заголовок документа. Элемент <title> не является частью документа и не показывается напрямую на веб-странице. В операционной системе Windows текст заголовка отображается в левом верхнем углу окна браузера (рис. 1). Допускается использовать только один тег <title> на документ и размещать его в контейнере <head>.

Рис. 1. Вид заголовка в браузере Internet Explorer

Рис. 1. Вид заголовка в браузере Firefox

Синтаксис
<head>
  <title>Заголовок</title>
</head>
